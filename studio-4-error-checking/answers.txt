Exercise answers 

1. What happens when the file does not exist?
When the file does not exist, the program prints:
"Error opening file: No such file or directory"
because open() fails.

2. What happens with perm.text after removing permissions?
When running the program on perm.txt after removing permissions, it prints:
"Error opening file: Permission denied"
because the file no longer has read permissions.

3. what does open() return on error? 
-1

4. if (fd == -1) is used to check error

6. return values and errno:

open  returns -1 on error, sets errno

read  returns -1 on error, sets errno

write  returns -1 on error, sets errno

close  returns -1 on error, sets errno

printf  returns a negative value on error, sets errno
